## 简介

Sketch 绘图工具是一款兼容 Vue 和 React 框架的跨平台组件，支持在小程序与 Web 环境中使用。使用 Sketch 不需要关注复杂的 canvas API，只需使用简单的组件化方式即可完成复杂的绘图需求。Sketch 组件化的设计理念使得开发者可以更专注于画布元素的实现，而不必担心底层实现细节。
